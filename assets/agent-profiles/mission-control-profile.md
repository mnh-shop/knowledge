## Auth Quick Reference

```bash
# Cookie login (for CLI -- stores session)
node mc-cli.cjs auth login --user admin

# API key auth via header
curl -H "x-api-key: $API_KEY" http://localhost:3000/api/tasks

# API key auth via Bearer
curl -H "Authorization: Bearer $API_KEY" http://localhost:3000/api/agents
```

Roles: `viewer` (read-only) < `operator` (manage tasks/agents) < `admin` (full access)

## CLI Quick Reference

```bash
node mc-cli.cjs <group> <subcommand> [options]

# Common commands
node mc-cli.cjs status health
node mc-cli.cjs agents list --json
node mc-cli.cjs tasks list --status inbox
node mc-cli.cjs events watch
node mc-cli.cjs sessions list

# Raw HTTP passthrough
node mc-cli.cjs raw --method GET --path /api/tasks
```

Exit codes: 0 success, 2 usage, 3 auth (401), 4 permission (403), 5 network, 6 server error.

## Deployment Quick Reference

```bash
# Local (auto-detects Docker if available)
bash install.sh

# Docker compose
docker compose up -d

# Standalone mode (no gateway)
docker compose --profile standalone up -d

# Hardened production
docker compose -f docker-compose.yml -f docker-compose.hardened.yml up -d

# Local standalone
pnpm start:standalone
```

Default port: 3000. Gateway port: 18789. Data directory: `.data/`.

## Env Quick Reference

```env
# Required (auto-generated by install.sh, or set manually):
AUTH_SECRET=<random>
API_KEY=<random>
AUTH_USER=admin
AUTH_PASS=<password>

# Gateway (optional -- omit for standalone mode):
OPENCLAW_GATEWAY_HOST=host.docker.internal
NEXT_PUBLIC_GATEWAY_OPTIONAL=true

# Direct API (gateway-free task execution):
ANTHROPIC_API_KEY=sk-ant-...
# or
OPENAI_API_KEY=sk-...
```

## Integration Summary

| System | Integration Method | Status |
|--------|-------------------|--------|
| OpenClaw | Gateway WebSocket + ACP protocol | Built-in (primary) |
| Hermes | Co-located filesystem + hooks + CLI | Built-in |
| n8n | Webhooks (outbound) + REST API | Via existing webhook system |
| Agentfield | Framework adapter pattern + API namespace | Not built-in, pattern documented |
| Claude Code CLI | Host-mounted binary + session continuation | Built-in |
| Any REST client | API key auth | Always available |

## MCP Server Quick Add

```bash

```

Config via `~/.mission-control/profiles/default.json` or `MC_URL`/`MC_API_KEY` env vars.
