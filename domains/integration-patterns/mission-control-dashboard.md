# Mission Control Dashboard

The Mission Control dashboard provides a web UI (port 3000) for monitoring agent health, displaying real-time status indicators, and showing webhook-pushed alerts. It polls OpenClaw's MCP server for agent metrics on a configurable interval.

See [[stack-reference-mission-control]] for the full deployment pattern.
