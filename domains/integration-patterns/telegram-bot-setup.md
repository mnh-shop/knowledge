# Telegram Bot Setup

Telegram bots serve as a primary user-facing channel in the deployment stack. Hermes forwards alerts to Telegram via bot token. OpenClaw agents can also interact with users through Telegram webhook handlers.

See [[stack-reference-mission-control]] and [[stack-reference-openclaw-n8n]] for bot token configuration.
