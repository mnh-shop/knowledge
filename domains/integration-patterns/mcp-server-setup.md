# MCP Server Setup

MCP server endpoints are configured via environment variables in deployment stacks. OpenClaw exposes its MCP server on port 18789, accessed by n8n and Mission Control for tool execution.

See [[stack-reference-mission-control]] and [[stack-reference-openclaw-n8n]] for environment variable configuration.
