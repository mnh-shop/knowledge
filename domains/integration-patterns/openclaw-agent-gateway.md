# OpenClaw Agent Gateway

OpenClaw serves as a central agent gateway in the deployment stack. It routes agent requests, exposes MCP tools, and provides webhook relay to downstream services like n8n and Mission Control.

See [[stack-reference-mission-control]] and [[stack-reference-openclaw-n8n]] for deployment patterns.
